# Mobile Asset Checklist

Place these files before store submission:

- `icon-1024.png` - iOS App Store icon, 1024x1024.
- `adaptive-icon.png` - Android adaptive foreground icon, at least 1024x1024.
- `splash.png` - Launch screen image, recommended 2048x2048 with safe padding.

Current repository only includes configuration and this checklist; final branded assets should be exported from design source and copied into this folder.

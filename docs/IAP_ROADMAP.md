# IAP Integration Roadmap

## Target
- Use RevenueCat SDK in Expo for StoreKit (iOS) and Google Play Billing (Android).

## Steps
1. Add RevenueCat package to `mobile/`.
2. Create products in App Store Connect and Google Play Console.
3. Map products and entitlements in RevenueCat.
4. Add backend webhook endpoint for entitlement sync.
5. Gate premium gameplay paths by entitlement status.
